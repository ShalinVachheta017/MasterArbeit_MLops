\section*{Pipeline Execution Paths}

The following three execution paths are supported.
Stages listed in parentheses are run only if the relevant flag is provided.

\begin{description}
  \item[Inference-only]
    Stages 1~$\to$~2~$\to$~3~$\to$~4~$\to$~5~$\to$~6~$\to$~7.
    Default when no flags are given.
  \item[Inference + Retraining]
    Stages 1--7 followed by 8~$\to$~9~$\to$(10 if \texttt{--update-baseline}).
    Triggered by \texttt{--retrain} or automatically when the trigger fires.
  \item[Full pipeline]
    Stages 1--10 followed additionally by 11 (\texttt{--calibrate}),
    12 (\texttt{--wasserstein}), 13 (\texttt{--adapt pseudo\_label}),
    14 (\texttt{--placement}).
\end{description}
